# Code of Conduct

This project adopts the [Contributor Covenant](https://www.contributor-covenant.org/version/2/1/code_of_conduct/), version 2.1.

In short: be respectful and constructive. Harassment, discrimination, and personal attacks are not tolerated in issues, pull requests, or any other project space.

## Enforcement

Report unacceptable behavior to **kang@opq.ooo**. Reports will be reviewed and handled confidentially. Maintainers may remove, edit, or reject contributions and comments that violate this code, and may ban repeat offenders.

The full text and enforcement guidelines are available at
<https://www.contributor-covenant.org/version/2/1/code_of_conduct/>.
